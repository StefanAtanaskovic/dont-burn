## Getting Started
To get started with the game, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/StefanAtanaskovic/dont-burn.git
   ```
2. Navigate into the project directory:
   ```bash
   cd dont-burn
   ```
3. Install necessary dependencies:
   ```bash
   npm install
   ```
4. Run the game:
   ```bash
   npm start
   ```
